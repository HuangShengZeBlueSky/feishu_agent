# 资源目录说明

更新时间：2026-04-25

## 目录约定

| 目录 | 放什么 | 给谁看 |
|---|---|---|
| `resources/mock/` | 模拟会议、群聊、文档、任务、日历、上下文清单、推进总表 | AI 和人做 mock 验收 |
| `resources/config/` | 环境变量模板、本地忽略密钥文件 | AI 找飞书配置入口 |
| `docs/` | 中文说明文档、进度、缺口清单、验证记录 | 人和 AI 快速理解项目 |
| `src/` | 代码实现 | AI 维护 |
| `output/` | 每次运行的黑盒输出结果 | 人验收，已忽略提交 |
| `run_logs/` | 每次运行的输入/输出/写入计划/错误记录 | 人和 AI 追踪执行，已忽略提交 |

## 飞书配置入口

| 文件 | 说明 |
|---|---|
| `resources/config/飞书环境变量模板.env` | 可提交模板，只有 App ID 和空 Secret |
| `resources/config/本地飞书密钥.env` | 本机真实密钥文件，已被 `.gitignore` 忽略 |
| `resources/config/本地模型secret.env` | 本机模型密钥文件，已被 `.gitignore` 忽略 |
| `resources/config/本地模型资源secret.json` | 本机模型清单和探测结果，已被 `.gitignore` 忽略 |

规则：真实 App Secret 只允许出现在本地忽略文件或当前 shell 环境变量里，不写进文档和源码。
