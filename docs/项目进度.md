## 当前实现方式

- `src/llm/client.py`：OpenAI 兼容 LLM 客户端。优先请求 JSON 模式；如果模型不支持 `response_format=json_object`，自动退回普通 chat 并解析 JSON。
- `src/core/llm_extractors.py`：用 LLM 抽取行动项、决策、风险，保留 `source_quote` 原文证据。
- `src/sources/feishu_project.py`：真实飞书 Base 数据源，负责读取上下文、行动项、决策、风险，并写入真实 seed/抽取结果。
- `src/workflows/real_examples.py`：真实业务 workflow。
- `src/adapters/bitable_adapter.py`：增加真实 Base 读取和按表写入能力。
- `tools/小龙虾飞书机器人入口.py`：飞书群自然语言入口，按 A/B/D 方向路由到真实 workflow，并在请求创建文档时调用飞书 Docs 创建真实文档。
- `C:/Users/黄胜泽/.agents/skills/xiaolongxia-project/SKILL.md`：OpenClaw 技能入口，让飞书机器人知道必须调用本仓库真实链路，不走 mock。

|---|---|---|---|
